# E-Commerce Back End

## Description
This application is the back end for an e-commerce web app, built with a MySQL2 database and the Sequelize package. The database includes product categories, products, and product tags.

## Installation 
Download the files from this repository, run the schema and seeds files to initialize and seed the database, and then type node server.js in the terminal. See the tutorial video.
